# Lingo Stream 5-Minute Demo Video Script

## 0:00-0:12 Opening Card
- On screen: `Lingo Stream` + team name + members.
- Narration/subtitle: "We are Team ___, and this is Lingo Stream, a Chrome extension for subtitle-based language immersion."

## 0:12-0:30 Core Idea
- Show one sentence before/after replacement.
- Narration/subtitle: "It detects YouTube captions in real time and replaces part of key words with inline translations."

## 0:30-1:45 Core Function Demo
- Open a YouTube video with captions ON.
- Open extension popup and set:
  - Provider: `Auto`
  - Language: e.g. `Spanish`
  - Mix `%`: e.g. `15%`
  - Enable + Save Words: ON
- Click `Save` and `Sync`.
- Show captions updating live on playback/scrub.
- Narration/subtitle: "The extension listens to caption mutations, filters low-value words, translates selected words, and renders inline translation instantly."

## 1:45-2:40 Reliability + Health
- Show `Health` panel:
  - Content script connected
  - Last translation success provider/time/count
- Click `Recheck`.
- Narration/subtitle: "We provide runtime health checks to confirm content-script connection and translation success status."

## 2:40-3:35 Vocabulary Workflow
- Show saved words table.
- Use search/filter briefly.
- Click `Export` (CSV), mention import/reset.
- Narration/subtitle: "Translated words can be stored locally, filtered, exported as CSV, or imported for continued learning."

## 3:35-4:35 Quiz Feature
- Click `Quiz`.
- Complete one matching round (5 pairs).
- Show score, accuracy, remaining words, next round.
- Narration/subtitle: "Lingo Quiz converts recent vocabulary into active recall practice with round-based matching and progress stats."

## 4:35-4:55 Ending
- On screen: recap bullets (real-time captions, configurable translation, vocabulary, quiz).
- Optional team appearance or thanks.
- Narration/subtitle: "Thank you for watching."

## Recording Checklist
- 1080p, stable screen capture, clear mic.
- Keep cursor movement deliberate.
- Use subtitles even with voice-over.
- Skip long setup/install footage; move quickly to live interaction.

## Quick Edit Version (One-Take, ~55s)

If you need a very fast cut, record this single paragraph:

"We are Team [Team Name], and this is Lingo Stream, a Chrome extension for YouTube language immersion.  
Lingo Stream detects caption updates in real time, keeps sentence context, and replaces selected words with inline translations.  
In the popup, users choose provider, target language, and mix percentage, then save and sync to apply changes instantly.  
The Health panel confirms content script connection and recent translation success data for runtime reliability.  
The vocabulary module stores translated words locally, supports filtering, CSV export, import, and reset.  
Finally, Lingo Quiz uses recent words in five-pair matching rounds and tracks score, accuracy, and progress.  
Lingo Stream combines real-time subtitle enhancement with vocabulary management and quiz-based reinforcement. Thank you."
